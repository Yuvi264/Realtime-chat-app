💬 Chat Application – Real-Time Messaging Platform

A full-stack chat application that allows users to send and receive messages instantly. It uses Socket.IO to enable real-time communication without refreshing the page.

📌 About the Project

This project is a real-time chat application where users can communicate with each other instantly.
It is built using React for the frontend and Node.js for the backend.

The main goal of this project was to understand how real-time communication works in web applications.

🚀 Features

🔐 User Signup and Login

💬 Real-time messaging (no page refresh)

👥 One-to-one and group chat

🟢 Online/Offline user status

🕒 Message timestamps

📱 Responsive design

🛠️ Tech Stack

Frontend: React, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB

Real-time: Socket.IO

⚙️ How It Works

User logs in

User sends a message

Message is sent to server using Socket.IO

Server sends the message to the other user instantly

👉 This makes the chat real-time without reloading the page

⚙️ Installation
# Clone the repository
git clone https://github.com/Yuvi264/chat-app.git

# Go to project folder
cd chat-app

# Setup backend
cd backend
npm install
npm run dev

# Open a new terminal and setup frontend
cd frontend
npm install
npm run dev
